# youtube-dorks
youtube dork queries for default file names

# Dorks
* IMG 3356
* IMG_0001
* DSC 1197
* WA_001 for random whattsapp videos
* .3gp
* AMBA0001 - AMBA9999 for dashcam videos
* Samsung have this naming convention: SAM_XXXX where X is a number
* MOV001 - MOV9999
* |
* ||
* .
* /'/
* /.-.
